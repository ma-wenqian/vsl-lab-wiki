# VSL现有打印机一览

该手册适用的对象应该是首次接触打印机或对打印机情况不熟悉的同学，所以提供的信息尽量实用、全面、细致，在不涉及技术问题的前提下(比如装料、打印机喷头堵了之类的)，初学者根据此手册能顺利完成打印，同时最好能显示出该打印机最优的性能。
